5. "Sistemas Operacionais" 

Sistema Operacional é o principal software do computador e faz a ponte entre o software e o hardware, permitindo que o usuário interaja os pormenores do computador. Ex: Linux, MacOS, Windows.
Pode-se usar Linux através do Windows como o WSL.
O SO é responsável por permitir que o usuário seja capaz de interagir com o hardware
Executar outros softwares, gerenciar o uso da memória, o armazenamento, o processamento, o sistema de arquivos, etc
Por exemplo, você sabe o que realmente acontece ao executar um arquivo no seu PC? 
Como os arquivos são armazenados em disco, o SO começará encontrando-o no discoO arquivo então é carregado para a memória RAM, que é mais eficiente que a memória secundária 
(HD, SSD, etc)Depois disso o arquivo será lido e executado pelo processador. Agora imagine isso acontecendo continuamente e em paralelo.


8. "Conhecendo o Terminal"

Utilizando o Terminal, usar ".." -> retrocede a pasta. Comando "cd" -> change directory, usa-se "cd Downloads" -> vai pra pasta download. Comando "ls" Lista o conteúdo da pasta atual.
Para acessar uma pasta ex: cd ../Leonardo (pode usar a barra/ e a virada pra esquerda também.)
Caso não se lembre o nome da pasta, clicar botão TAB pra completar o nome da pasta que quer ir. -> code . -> cria arquivo na pasta raiz "C" lá no VSCODE
"cp" -> copia arquivo e o nomeia -> cp index.html ./home.html
"mv" -> move arquivo -> mv home.html ../ <- Manda pra pasta anterior
"rm" -> remove arquivo ou diretório
